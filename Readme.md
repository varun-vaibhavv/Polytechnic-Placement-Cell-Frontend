# Placement and Training Cell Website  
Government Polytechnic, Muzaffarpur

## Overview
This project was developed as the **major project during Semester V of my
polytechnic diploma**. The objective was to design and implement the
front-end of a Placement and Training Cell website for an academic institution.

## Project Scope
The work focused exclusively on **front-end development**, including:
- Page layout and structure
- Navigation and user interface design
- Presentation of placement- and training-related information

## Features
- Static and responsive web pages
- Clear navigation for students and faculty
- Structured presentation of placement and training details
- Institutional-style design suitable for academic use

## Technologies Used
- HTML
- CSS
- Bootstrap
- JavaScript

## Role and Contribution
- Designed and implemented the complete front-end of the website
- Structured pages for clarity and usability
- Ensured consistency in layout and visual presentation
